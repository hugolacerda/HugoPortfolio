== Description ==

Learn about Tidio Gallery a beautiful and functional Gallery which you can easily place on your website. Full integration with mobile devices, beautiful design, real-time saving - are only a part of its functionalities.

[youtube http://www.youtube.com/watch?v=6Bx0Fx89FI8]

= Functionalities: =

* Beautiful design 
* Possibility of placing the gallery at any place in the website 
* Autosave - all changes are recorded in real time, so you are not going to lose the effects of your work 
* Mobile ready
* Completely free!

== Installation ==
1. Go to WordPress Control Panel
1. Click "Plugins", then "Add New"
1. Type in "Tidio Gallery" and click "Search Plugins"
1. Download and install the Plugin
1. Click the "Activate Plugin" link
1. Move to the "Visual Editor" section in the menu
1. Click "Begin Website Editing"
1. You will be redirected to the editor. Done!
